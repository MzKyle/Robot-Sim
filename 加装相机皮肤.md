- 资源接入：
  - 复制 `mdoel_camera/model.config`、`model.sdf`、`meshes/3dCamera.DAE`、`materials/textures/3dCamera.png` 到 `src/data_collect_sim/models/rgbd_camera/`。

  - `CMakeLists.txt` 已安装整个 `models` 目录，无需新增安装规则。
- 末端相机模型：
  - 在 `src/data_collect_sim/models/panda_weld_arm/model.sdf` 的 `camera_mount` link 中，删除当前 `0.08m` 蓝色 box visual/collision。
  - 加入相机 mesh visual：
    - URI: `model://rgbd_camera/meshes/3dCamera.DAE`
    - pose: `0 0 0 0 0 1.5707963267948966`
    - 目标效果：外观镜头面朝向 `camera_mount` 的 `+X`，与两个仿真传感器成像方向一致。
  - 使用相机模型的碰撞和惯性尺寸作为基础，碰撞可用简化 box，随同样 yaw 旋转；质量使用 `0.3kg`，惯性使用外部模型中已有值。
- 传感器与坐标：

  - `data_collect_sim.launch.py` 中 `tool_frame` 继续为 `panda_weld_arm/camera_mount`。
