# Code of Conduct

本项目采用简洁的贡献者行为准则。

## 我们的承诺

我们希望社区交流保持友善、专业、聚焦事实。不同经验背景的贡献者都应能安全地提出问题、报告缺陷和提交改进。

## 期望行为

- 使用尊重、包容的语言。
- 讨论代码、配置和实验结果，而不是攻击个人。
- 清楚描述问题、复现步骤和约束。
- 接受维护者为保持项目方向所做的取舍。

## 不接受的行为

- 人身攻击、骚扰、歧视性表达。
- 发布他人的隐私信息。
- 恶意刷屏或持续偏离主题。
- 明知有害仍发布破坏性操作说明。

## 执行

维护者可以编辑、隐藏或删除违反准则的 issue、PR、评论，也可以临时或永久限制相关账号参与项目。
