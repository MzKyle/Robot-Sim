# 桌面操作台

`data_collect_ui` 是与采集后端配套的桌面界面，用于查看状态、录入任务、浏览历史数据和调整参数。

## 页面组成

- `采集操作`：查看 ROS、采集、Fanuc 和质量状态，录入任务信息并控制启停。
- `实时预览`：显示 2D 图像、3D 点云预览和关键统计信息。
- `历史数据`：扫描数据目录下的 `manifest.json`，展示采集记录并支持打开目录。
- `参数设置`：编辑 `nodemanage.yaml`，修改后自动保存。

## 交互特点

- 支持自动保存配置。
- 支持按钮控制采集启停、3D 扫描、配置重载和任务信息保存。
- 支持历史记录扫描和刷新。
